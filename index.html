Formula Reference

RPM / Feed

Formula: RPM = (SFM × 12) / (π × D)

Feed Formula: Feed Rate = Chip Load Per Tooth × Number of Flutes × RPM

Reference: Machinery's Handbook — Speed and Feed calculations (Industrial Press)

Code: calcRpm(), calcFeed() in index.html

Tapered Hole Offset

Formula: Radius Change = (Target Ø − Current Ø) / 2

Formula: Z Move = |Radius Change| / tan(Taper Angle per Side)

Note: Taper angle is entered in degrees.

Convention: Haas vertical — Z− (down) increases hole diameter when radius change is positive.

Reference: Machinery's Handbook — Tapers and Taper Calculations (Industrial Press)

Code: calcTaper() in index.html

X/Y Hole Location

Formula: Edge Position = Nominal Position − (Hole Diameter / 2)

Reference: Machinery's Handbook — Jig Boring and Hole Location (Industrial Press)

Code: xyStep3() in index.html
